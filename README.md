# Paradise Nursery

A React-based shopping cart application for an online plant shop. Browse houseplants by category, add them to your cart, adjust quantities, and manage your shopping experience.

## Features

- Browse plants organized by categories (Air Purifying, Aromatic, Indoor Flowering)
- Add plants to shopping cart with a single click
- View cart with item details, quantities, and total cost
- Increase, decrease, or remove items from cart
- Responsive navbar with cart item count indicator
- Landing page with company introduction
- About Us page with company details

## Tech Stack

- React (Vite)
- Redux Toolkit
- React Router
- CSS3

## Getting Started

```bash
npm install
npm run dev
```
